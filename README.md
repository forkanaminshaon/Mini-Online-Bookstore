# Mini Online Bookstore

This project was developed for the Rokomari Growth Analytics & CRO Specialist assignment.

## Features

Book listing page  
Book details page  
Search books  
Add to cart simulation  
REST API integration  

## Tech Stack

Frontend
HTML
CSS
Vanilla JavaScript

Backend
Spring Boot

Database
MySQL

## Run Backend

mvn spring-boot:run

## Run Frontend

Open index.html with Live Server
